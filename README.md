# Space-X-Capstone-Project
IBM Data Science Professional Certificate capstone project analyzing SpaceX Falcon 9 launch data using data visualization, SQL, interactive dashboards, and machine learning.

This repository contains the IBM Data Science Professional Certificate capstone project focused on SpaceX Falcon 9 launch analysis and landing prediction.

* Project Components:
Data Collection using SpaceX API and Web Scraping
Data Wrangling and Cleaning
Exploratory Data Analysis (EDA)
SQL Analysis
Interactive Visual Analytics with Plotly Dash
Folium Interactive Maps
Predictive Machine Learning Models

* Tools & Technologies:
Python
Pandas
SQL
Plotly Dash
Folium
Scikit-learn
BeautifulSoup

* Objective:
The goal of this project is to analyze Falcon 9 launch data and predict first stage landing success using classification models.
